# 🏛️ Knovault（芝士庫）

把**數位書庫、實體書櫃、筆記軟體**三個知識孤島整合進同一個安全金庫的自託管個人知識管理（PKM）系統。以「書」為知識源頭節點，無縫解析數位書、將實體書數位孿生化，並結合卡片盒雙鏈筆記與多模態圖片資產。

## 技術選型

- **後端**：.NET 8/9 (C#) · ASP.NET Core Minimal API · EF Core
- **資料庫**：SQLite（領域層抽象，日後可換 PostgreSQL）
- **前端**：Vue 3 · Vite · Naive UI（PWA 可安裝成桌面視窗）
- **形態**：雙擊執行的 Windows 程式 → 本機 Kestrel 伺服器 + 瀏覽器/PWA 介面；同網段裝置可連；未來可 Docker 上 NAS
